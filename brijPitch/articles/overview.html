    Product: Overview

    The Skinny

    • The first truly compleat, Enterprise-ready, easy-to-use No-Code Database System in the world.
    • As powerful as most Low-Code Systems. So simple to use that any Spreadsheet user can easily create new Tables, build Forms and execute Queries without writing a single line of code.
    • Infinite Capacity.
    • Nothing in the market today comes close.
    • 80% complete

    The Details

    80% Complete?

    Yes. We began work on this project in 2016.
    We have invested over $2M in Sweat Equity since.
    The intention was for the Development to be self-funded (zero Venture Capital).
    Unfortunately, the development process took much longer than expected and we have run out of working capital.

    Why did it take longer than expected?

    • Please take a quick look at the Project Timeline before reading below.
    • Trying to do things no other company has done before: a combination of Low-code and No-code systems.
    • About midway through the project the Founder discovered that this product would need MetaProgramming (click here for details)
    • The technical challenges in developing such a product are not even possible to overcome using OOP methods (the Founder was then using Java as the back-end). Moreover, even though languages like C++ offer Template MetaProgramming, they still weren't sufficient: the only answer was to use a Functional Language, and blaze a new path.
    • Brij uses very cutting edge technology - MetaProgramming constructs in a purely Functional paradigm - which is firmly in the research realm - this technology is used also in Quantum Computing (details here). Had these constructs and abstractions even been available in OOP, other vendors would've built something close to Brij. We have a very clear and definite edge.
